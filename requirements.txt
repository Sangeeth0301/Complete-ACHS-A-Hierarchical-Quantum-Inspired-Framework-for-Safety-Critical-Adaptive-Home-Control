# Complete ACHS: Project Requirements & Tech Stack

## 🖥️ Computing Requirements
### Fog Node (Main Brain)
- **Hardware:** Raspberry Pi 5 (8GB RAM Recommended).
- **Storage:** M.2 NVMe SSD (for high IOPS telemetry logging).
- **OS:** Raspberry Pi OS 64-bit (Debian Bookworm).

### Edge Nodes
- **SoC:** ESP32-S3 Dual-Core (with Hardware Crypto Acceleration).
- **Protocols:** WiFi 6 / BLE 5.0.

## 🔌 Hardware Sensor Manifest
- **Climate:** DHT22 (High-accuracy Temperature & Humidity).
- **Light:** BH1750 (Digital I2C Lux Sensor).
- **Motion:** HC-SR501 PIR Sensor.
- **Safety:** MQ-5/MQ-9 (LPG/Natural Gas Sensor).
- **Actuators:** Solid State Relays (SSR) - *Mandatory for Spark-Prevention*.

---

## 🔐 Security Requirements
- **Mutual TLS (mTLS 1.3):** X.509 Device Certificates.
- **Encryption:** AES-256 (Hardware accelerated on ESP32).
- **Identity:** Local CA (Certificate Authority) setup on Fog node.
